# Mini HR Vanilla

HR management system for small companies (10-20 employees).

## Stack
- PHP 8+ | Vanilla JS | CSS custom | JSON flat files

## Setup
1. Copy to PHP 8+ server
2. Set permissions: `chmod -R 770 data/`
3. Open `/login.php` in browser
